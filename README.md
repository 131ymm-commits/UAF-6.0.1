## Главный результат (унаследован из v5.2, выведен из мастер-уравнения)

Водораздел коллапса сети определяется ведущим собственным значением:

```
A0_crit = K0 / Re(λ_max),     K0 = (δ−f)/α_s = 0.167
```

- `λ_max` — ведущее собственное значение матрицы смежности
- Это структура эпидемического порога (класс универсальности SIS)
- Выведено из мастер-уравнения (v5.2, Q35), не подгонка
- Универсально по доменам: биология, инфраструктура, метаболизм (ошибка ≤20%)

Полный вывод и история — `UAF_v5.2_FINAL_COMMUNIQUE_inherited.txt`.

## Мастер-уравнение

```
dA_i/dτ = α_s·C_ij·A_j·(1−A_i) + α_l·Π_i·PE_i·(1−A_i)
        + f·(1−A_i) − δ·(1−0.3·A_i)
```
`α_s=0.06, α_l=0.01, f=0.002, δ=0.012`, `A_i ∈ [0,1]`.

## Управляющая величина (v6.0)

```
A0_crit = K0 / Re(λ_max)
```
- беззнаковые сети: Re(λmax)=λmax (теорема Перрона-Фробениуса)
- направленные: Re ведущего (возможно комплексного) собственного значения
- знаковые, >40% торможения: Im(λmax)≠0 → колебательный режим
