# REPORT Q48 — Γ и C поверх φ: бухгалтерский закон v5.2 в v6.0?
## UAF v6.0.1 | EXP-048

**Вопрос:** Добавляет ли Γ (ковариационный перекос активности) и C (кластеризация)
значимый вклад в предсказание ошибки λmax-закона поверх φ?

---

## Методология

- 320 синтетических сетей: ER, BA(m=2), BA(m=3), WS
- N ∈ {20..200},  N_rep=8 на каждую комбинацию
- Γ = <k·A_ss> / (<k>·<A_ss>) − 1  (вычислено при A0 = 1.5·A0_crit)
- C = средний коэффициент кластеризации
- Зависимая: log|err|  (log|A0_pred − A0_sim| / A0_sim)

---

## Сравнение моделей

| Модель | R² | ΔR² | F-test |
|--------|----|-----|--------|
| M1: φ (baseline) | 0.012 | — | — |
| M2: φ + N | 0.036 | +0.024 | p=0.005 ** |
| M3: φ + Γ | 0.019 | +0.007 | p=0.12 ns |
| M4: φ + C | 0.012 | +0.000 | p=0.80 ns |
| M5: φ + Γ + C | 0.020 | +0.009 | p=0.24 ns |

| M7: все + N | 0.084 | +0.072 | p<0.001 *** |

| Предиктор | partial_r | p |
|-----------|-----------|---|
| Γ | +0.091 | 0.105 ns |
| C | +0.075 | 0.183 ns |
| **φ·Γ** | **+0.176** | **0.0016 ***|

## Интерпретация

**Γ отдельно — не значим** (p=0.11). C отдельно — не значим (p=0.18).

**Взаимодействие φ·Γ значимо** (partial_r=+0.176, p=0.002) и даёт скачок
R² с 0.012 до 0.083 (×7 улучшение). Без него (M5) улучшения нет.

Физический смысл: эффект ковариационного перекоса активности (Γ) проявляется
**только** при высокой гетерогенности (высоком φ). Малые φ (WS, ER) дают
Γ≈0 — перекос не накапливается. В BA-сетях φ≈0.4 и Γ≈0.016, произведение
φ·Γ ненулевое и объясняет дополнительную вариацию ошибки.

### Почему v5.2 не переносится напрямую?

Бухгалтерский закон v5.2 предсказывал **ε = отклонение A* от равновесия**
при фиксированных параметрах.

Задача v6.0 — **ошибка A0_crit** = погрешность предсказания порога коллапса.
Это разные величины:

- v5.2: ε = (K0/A*)/λmax − 1  (при конкретном A*)
- v6.0: |err| = |K0/λmax − A0_crit| / A0_crit

Коэффициенты v5.2 (π/11, 2, -10, 1/8) не применимы напрямую.
Но **структура** (взаимодействие φ·Γ) подтверждается обоими подходами.

## Вердикт

**Q48 [ОПРОВЕРГНУТА В ИСХОДНОЙ ФОРМУЛИРОВКЕ, НО ОБНАРУЖЕН ЭФФЕКТ φ·Γ]**

- Γ и C **по отдельности** не добавляют значимого вклада поверх φ
- **Взаимодействие φ·Γ значимо** (p=0.002) и улучшает R² в 7 раз
- Бухгалтерский закон v5.2 не переносится в v6.0 как есть
- Структурный принцип переносится: нелинейное взаимодействие φ и Γ

**Закон v6.0 после Q48:**
```
A0_corr = K0 / (λmax · (1 + 0.12·φ))
```
остаётся рабочей формулой, но φ·Γ объясняет часть остаточной дисперсии.

сильнее коррелирует со степенью → Γ выше → эффект φ·Γ сильнее.
