# REPORT Q46 — φ-механизм ошибки λmax-закона
## UAF v6.0.1 | EXP-046

**Гипотеза:** `|err| ~ φ = Var(k)/<k²>` — крупные сети гетерогеннее,
φ-остаток из v5.2 Q37 растёт → ошибка растёт. Связь v6.0 с v5.2.

---

## Методология

- 112 синтетических сетей: ER (φ≈0.13), BA (φ≈0.47), WS (φ≈0.02), Mixed (φ≈0.44)
- N ∈ {20, 30, 50, 80, 120, 200, 300, 500}
- `|err| = |A0_pred − A0_sim| / A0_sim`
- `A0_pred = K0/λmax`,  `K0 = 0.167`
- Симуляция (N≤200): бисекция по порогу коллапса
- Контроль (N>200): mean-field поправка
- Частные корреляции Пирсона на log-шкале

## Интерпретация

**φ — первичный предиктор** (partial_r=+0.459 > N partial_r=+0.323).

Механизм (Q37): λmax-закон выведен в среднеполевом приближении,
игнорирующем флуктуации степеней. Точный закон:
```
A0_crit = K0/λmax · [1 + O(φ)]
```
φ = Var(k)/<k²> количественно измеряет этот остаток.
BA-сети (высокое φ) → ошибка втрое выше WS-сетей (низкое φ).

**N не конфаундер, а вторичный фактор:** крупные сети статистически
гетерогеннее (ρ(N,φ)=+0.19), но при фиксированном φ размер сам по себе
добавляет ошибку (partial_r=+0.323). Возможный механизм: конечно-размерные
эффекты в среднем поле.

R²=0.300 — ниже Q45 (R²=0.52). Объяснение: Q45 работал с реальными
доменами (там N и φ сильнее скоррелированы). На синтетике с
контролируемым φ — чище, но меньше дисперсии объяснено.

## Вердикт

**Q46 [ДЕРЖИТСЯ, φ-МЕХАНИЗМ ПОДТВЕРЖДЁН]**

φ = Var(k)/<k²> — первичный предиктор ошибки λmax-закона
(partial_r=+0.459, p<0.0001), опережает N при раздельном контроле.
Это связывает v6.0 с v5.2: остаток [1+O(φ)] из Q37 не просто
формальный — он количественно определяет область применимости закона.

**Практическое следствие:**
Перед применением λmax-закона к реальной сети — вычислить φ.
При φ > 0.4 (BA-подобные) ожидаемая ошибка ≥5%.
При φ < 0.1 (WS/решёточные) — ошибка ≤2%.
