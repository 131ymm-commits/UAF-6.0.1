# C. elegans Connectome — NeuronConnect.xls

**Источник:** WormAtlas (формат Varshney et al. 2011 / Cook et al. 2019).
**Содержание:** 6417 связей, 283 нейрона.

## Формат
Колонки: `Neuron 1`, `Neuron 2`, `Type`, `Nbr`
- `S`/`Sp` — химический синапс Neuron1 → Neuron2 (направленный)
- `R`/`Rp` — приём (реципрокная запись S, отбрасывается как дубль)
- `EJ` — щелевой контакт / gap junction (ненаправленный)
- `NMJ` — нейромышечный (отбрасывается, не нейрон-нейрон)
- `Nbr` — число контактов (вес)

## Извлечённая статистика (ненаправленная, бинаризованная)
- N_eff = 279 связанных нейронов
- <k> = 16.41
- χ = <k²>/<k> = 25.94
- het = <k²>/<k>² = 1.58
- clustering C = 0.22
- max degree = 93

## Использование
EXP 022 (Q22): валидация het-поправки на реальной топологии.
Результат: χ·A0 = 0.143, формула Q21 предсказывает 0.146 (ошибка −2.2%).
