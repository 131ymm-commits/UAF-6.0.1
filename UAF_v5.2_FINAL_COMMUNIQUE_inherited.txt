ОДНОЙ СТРОКОЙ:
  Водораздел коллапса сети — СПЕКТРАЛЬНЫЙ порог A0=(δ−f)/(α_s·λmax),
  выведен из мастер-уравнения, универсален по доменам, в классе
  эпидемических порогов. Динамика реальных нейронов НЕ операционализируется
  (проверено 4 способами). Обе границы — твёрдые и честные.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ГЛАВНЫЙ РЕЗУЛЬТАТ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   ┌──────────────────────────────────────────────────────────┐
   │   A0_crit = (δ−f)/(α_s·λ_max) · [1 + O(φ)]               │
   │                                                          │
   │   λ_max = ведущее собств. значение матрицы смежности     │
   │   φ = Var(k)/<k²> = 1−1/het ∈ [0,1)                      │
   │   (δ−f)/α_s = 0.167 — ВЫВЕДЕНО из мастер-уравнения        │
   └──────────────────────────────────────────────────────────┘

  • A0=K0/λmax ВЫВЕДЕНА (Q35): гомогенный предел → K0=(δ−f)/α_s без
    подгонки; линейная устойчивость нижнего состояния → <k> заменяется
    на λmax (структура эпидемического порога).
  • Внешне подтверждено сравнением с SIS (Q38): UAF в классе
    универсальности λmax-порогов (SIS, bootstrap-перколяция, каскады).
  • Точна при φ=0 (регулярные/ER). Ошибка ≤20% иначе.
  • Поправка [1+O(φ)] — ограниченный нелинейный остаток (насыщение хабов),
    БЕЗ замкнутой формы (Q37). Граница выводимости, не пробел.

  Проверка на доменах (старая формула Q21 давала +171% на метаболизме):
    Метаболизм E.coli: −1..−4% | Энергосеть: ±0% | Белки: +4% | C.elegans: −3..−5%
