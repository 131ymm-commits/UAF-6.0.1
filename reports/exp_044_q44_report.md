# UAF v6.0 — EXP 044 / Q44
# Закон на истинно асимметричных сетях — граница по цикличности

## Контекст
Q42 (двудольные экосети, 6%) и Q43 (направленные пищевые после симметризации,
5.5%) показали: закон A0=K0/λmax держится на экологии. Q44: выходит ли он за
пределы экологии на направленные сети других доменов, и какая спектральная
величина работает при асимметрии?

## Данные
Network_Data_MJS20 (Samuel Johnson), 6 доменов, асимметрия 0.08–1.0:
торговля (Trade), социальные (Social), язык (Language), коннектомы (Neural),
генная регуляция (Genetic), метаболизм (Metabolic). Реальные данные к
PNAS/Nature статьям. K0=(δ−f)/α_s=0.167 выведена, не подгоняется.

## Механизм границы (связь с Q43)

Доля обоюдных связей (2-циклов) определяет применимость:

| Сеть | 2-цикл.доля | Базальных% | Результат |
|------|------------|-----------|-----------|
| gene_coli | 0.00 | 18% | коллапс |
| gene_yeast | 0.00 | 14% | коллапс |
| metab_CQ | 0.02 | 7% | работает (−6%) |
| celegans | 0.17 | 9% | работает |
| trade | 0.74 | 0% | работает |

Генные сети — чистая иерархия регуляции (0 циклов) + много базальных узлов →
коллапс в ABM, тот же механизм, что направленные пищевые сети до
симметризации (Q43). Закон требует структурной замкнутости.

## Выводы

1. **Закон выходит за пределы экологии.** Метаболизм (−6%), торговля
   (σmax −0%), нейро (celegans σmax +8%) — работает на циклических
   направленных сетях разных доменов с ошибкой 5–8%.

2. **Нет универсальной спектральной величины.** σmax обычно лучше Re(λmax)
   при асимметрии, но взрывается на отдельных сетях (rhesus2 +142%). Единой
   формулы для асимметрии нет — открытый вопрос (как остаток φ в v5.2 Q37:
   ограниченный эффект без замкнутой формы).

3. **Граница структурная — по цикличности.** Ацикличные иерархии потока
   (генная регуляция) вне области применимости: там нет бистабильного
   водораздела вообще. Это свойство динамики UAF, не недостаток величины.
   131ym: "их круг не замкнут" — точный диагноз и для пищевых сетей (Q43),
   и для генной регуляции (Q44).

## Вердикт Q44: ЧАСТИЧНО ДЕРЖИТСЯ + ГРАНИЦА ПО ЦИКЛИЧНОСТИ

| Проверка | Результат |
|----------|-----------|
| Закон вне экологии (метаб/торговля/нейро) | работает, 5–8% |
| K0 подгонялась? | нет, выведена (v5.2) |
| Ацикличные иерархии (генная регуляция) | коллапс, вне области |
| Спектральная величина при асимметрии | σmax чаще лучше, но не универсум |
| Граница | структурная (замкнутость), согласована с Q43 |

Закон обобщается на циклические направленные сети многих доменов, но требует
наличия циклов. Выбор спектральной величины при асимметрии — открытый вопрос.
