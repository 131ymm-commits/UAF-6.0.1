# UAF v6.0 — EXP 045 / Q45
# Количественный закон применимости: ошибка vs цикличность

## Контекст
Q43/Q44 показали механизм: закон A0=K0/λmax требует структурной замкнутости
(ацикличные сети — генная регуляция, направленные пищевые без симметризации —
коллапсируют). Q45: есть ли КОЛИЧЕСТВЕННАЯ связь точности с цикличностью?

## Данные
53 реальные сети, 6 доменов (MJS20): пищевые (symmetrized), торговля,
социальные, нейро, метаболизм, язык. N=15–500. Мера цикличности = доля
2-циклов (обоюдных связей) в исходной направленной матрице. K0 выведена.

## Результаты (реальный запуск, 53 сети)

Сырая корреляция cyc↔|err| = **−0.51** (больше циклов → меньше ошибка).
Но размер сети — confound. Множественная регрессия |err| ~ N + cyc:

| Фактор | Коэффициент | Частная корреляция |
|--------|-------------|--------------------|
| Размер N | +1.60 | **+0.67** (главный) |
| Цикличность | −2.87 | −0.31 (реальный, слабее) |
| | | R² = 0.52 |

Примеры:
- торговля (cyc=0.74, N=24): ошибка 0.8% — циклично, мало
- celegans (cyc=0.17, N=297): ошибка 12.7% — ациклично, крупно
- пищевые symmetr. (cyc=0, N=48–349): 4–9% — работают через симметризацию

## Выводы

1. **Цикличность реально снижает ошибку** (частная корр −0.31 после контроля
   размера) — подтверждает механизм Q43/Q44 (закон требует замкнутости).

2. **Но размер сети — доминирующий фактор** (частная корр +0.67): крупные
   сети дают больший разброс.

3. **Единого чистого закона "ошибка = f(цикличность)" нет.** Зависимость
   многофакторна, R²=0.52 — половина дисперсии объяснена, половина нет.

## Вердикт Q45: ЧАСТИЧНО ПОДТВЕРЖДЕНО

Честно: связь "ошибка~цикличность" реальна, но не доминирует. Это не новый
закон применимости, а количественное подтверждение механизма Q43/Q44 +
обнаружение, что размер сети влияет сильнее. Согласуется со стилем v5.2
(Q37): реальные эффекты часто ограничены и без замкнутой формы.

| Проверка | Результат |
|----------|-----------|
| Цикличность снижает ошибку? | да, частная корр −0.31 |
| Главный фактор? | нет, размер сети (+0.67) |
| Единый закон применимости? | нет, R²=0.52, многофакторно |
| K0 подгонялась? | нет, выведена |

## Следующий вопрос Q46
Почему размер увеличивает ошибку? Гипотеза: крупные сети гетерогеннее (выше
φ=Var(k)/<k²>), а φ-остаток (v5.2 Q37) растёт с гетерогенностью. Проверить
|err| ~ φ напрямую. Если φ объясняет размерный эффект — это свяжет v6.0
(домены) с v5.2 (остаток φ) в единую картину. Данные есть (MJS20).
