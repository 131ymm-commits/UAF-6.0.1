## Ключевое наблюдение: систематичность
Ошибка СТАБИЛЬНО отрицательная (−2.7% до −10.5%), не разброс вокруг нуля.
Закон консистентно чуть недооценивает водораздел. Систематическое смещение =
закон ловит реальную структуру, а смещение объяснимо: двудольность даёт
запас устойчивости (реальные экосети чуть устойчивее спектрального
предсказания). Случайный разброс означал бы шум; систематика означает закон.

## φ-поправка не переносится (проверено на исходных 3)
φ-поправка Q37 (калибр. на BA/ER) на двудольных ухудшает (+13..16%).
Leading-форма лучше. Подтверждает Q37: поправка класс-специфична, leading
K0/λmax фундаментальна и переносима на новый класс топологии.

## Интерпретация для экологии
A0_crit — критическая доля выживающих видов, ниже которой сеть коллапсирует
к вымиранию. Закон даёт порог из структуры через λmax. Систематическое
занижение → двудольная структура взаимодействий даёт запас прочности.

## Вердикт Q42: ДЕРЖИТСЯ (твёрдо, 8 сетей)
- A0=K0/λmax на 8 экосетях, средняя ошибка 6%, без подгонки
- Систематическое (не случайное) смещение → закон отражает реальность
- Двудольная структура — новый класс топологии, leading-форма переносима
- φ-поправка не нужна (подтверждает Q37)

Первое практическое применение закона, выборка достаточна для вывода.
Спектральный порог универсален по структуре для двудольных экосетей.

## Граница (см. Q43)
На НАПРАВЛЕННЫХ трофических сетях (FW_005, Gulf of Cadiz) закон отклоняется
(−42%). Двудольные (симметричны по построению) работают; сильно-асимметричные
трофические — нет. Разница в асимметрии.
