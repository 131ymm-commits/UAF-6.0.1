# UAF v6.0 — EXP 041 / Q41
# Комплексный λ_max и теорема Перрона-Фробениуса

## Контекст
Q40 показал, что закон держится для направленных сетей через Re(λmax), но
комплексный λmax не возник. Q41: когда λmax комплексный, что делает Im(λmax)?

## Главное открытие (не ожидалось): Перрон-Фробениус
Для НЕОТРИЦАТЕЛЬНОЙ матрицы (все C_ij≥0) ведущее собственное значение
(по модулю) ВЕЩЕСТВЕННО. Комплексные собственные значения существуют, но
никогда не ведущие. → Для любой беззнаковой сети Re(λmax)=λmax всегда.
Вопрос «комплексный λmax» не имеет смысла для беззнаковых сетей.

Комплексное ведущее λmax требует ОТРИЦАТЕЛЬНЫХ связей → ЗНАКОВЫХ сетей.
Это связывает Q41 с Q39.

## Результаты (знаковые сети)

### Спектр vs торможение
| Торможение | Re(λlead) | Im(λlead) |
|-----------|-----------|-----------|
| 0% | 6.00 | 0.00 |
| 30% | 2.79 | 0.00 |
| 40% | 2.45 | +0.27 |
| 60% | 2.59 | +0.99 |

Комплексный ведущий λmax — только при сильном торможении (>40%).

### Закон с дрейфом
| Торможение | A0·Re(λmax) |
|-----------|-------------|
| 0% | 0.166 |
| 10% | 0.155 |
| 20% | 0.140 |
| 30% | 0.140 |

Re(λmax) — правильная величина даже для знаковых, с поправкой от торможения.

### Осцилляции
При Im(λmax)=0.99 (60% торможения) траектория средней активности дала
25 смен знака производной → ОСЦИЛЛЯЦИИ. Im(λmax) → колебательное
приближение к аттрактору. Гипотеза подтверждена.

## Связь с Q39
Реальный знаковый коннектом C. elegans — только 3.9% тормозных связей,
далеко от комплексного режима (>40%). Его λmax вещественно, закон не
меняется. Это объясняет, почему знаки в Q39 не повлияли на структурный
закон: реальное торможение слишком разрежено, чтобы сдвинуть спектр.

## Вердикт Q41: ДЕРЖИТСЯ + теоретическое прояснение
- Перрон-Фробениус: Re(λmax) хорошо определён для беззнаковых сетей
- Закон робастен для направленных и слабо-знаковых
- Комплексный λmax только при >40% торможения → осцилляции
- Объясняет Q39: реальное торможение слишком разрежено для эффекта

Спектральное обобщение завершено: закон корректен для направленных и
знаковых сетей, граница комплексного режима установлена.
